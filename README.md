# Русский лендинг «Мещанского урбеча»

Это черновой статический лендинг для русской версии книги.

## Назначение

- дать одну понятную ссылку для фестиваля «ДВОР»;
- использовать как ссылку в заявке «Твоя Полка»;
- заменить разрозненный путь через Taplink, когда нужно показать именно книгу;
- вести читателя к покупке на Ridero/Ozon/Wildberries и к отзывам на LiveLib.

## Файлы

- `index.html` - страница;
- `styles.css` - оформление;
- `assets/cover-ru.jpg` - русская обложка;
- `assets/prologue.png` - иллюстрация пролога;
- `assets/what-to-do.png` - иллюстрация финальной главы.
